Create a jaw-dropping, cinematic, AAA-quality, single-page interactive website called "Winter Cipher" (or another original medieval fantasy title). The purpose of the website is to decode an encoded message using the following exact algorithm and NEVER change or simplify it:

Algorithm (must remain the core functionality):

Read the entire encoded message.
Find the first word.
Count the number of alphabetic letters in the first word.
Use that count as the shift value.
Shift every alphabetic character in the entire message backward by that value using Caesar Cipher logic.
Preserve spaces, punctuation, and numbers.
Support uppercase and lowercase without losing their case.
Display the detected shift value and visually explain how it was calculated.
Animate the transformation of every letter during decoding so users understand the algorithm rather than simply seeing the answer.
